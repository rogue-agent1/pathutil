# pathutil
File path manipulation: dirname, basename, ext, glob, exists, rename preview.
```bash
python pathutil.py basename /usr/local/bin/python3
python pathutil.py ext file.tar.gz
python pathutil.py glob "**/*.py"
python pathutil.py exists /etc/hosts ~/.bashrc
python pathutil.py rename "\.txt$" ".md" *.txt
```
## Zero dependencies. Python 3.6+.
